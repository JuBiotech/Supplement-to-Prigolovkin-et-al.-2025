This repository contains the data, code and manuscript of *Turbid but Accurate: Automating Lysostaphin Quantification including Uncertainty Quantification* (2025), L. Prigolovkin, M. Osthege, M. Siska, J. Sander, A. Hoffzimmer, W. Wiechert, C.K. Desiderato, C.U. Riedel, M. Oldiges.

# Contents
- `/data` contains the raw experimental data
- `/manuscript` includes the manuscript figures and additional supporting figures not part of the manuscript
- `/code` contains the Opentrons OT-2 turbidity assay script for measurement of 8 samples
